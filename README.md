# First Flutter App – Лабораторная работа №2

Простое демонстрационное Flutter-приложение, созданное в рамках знакомства с фреймворком. Приложение выводит приветствие "Hello world!" на градиентном фоне и демонстрирует базовые виджеты: `MaterialApp`, `Scaffold`, `Container`, `Center`, `Text`, а также работу с `BoxDecoration` и `LinearGradient`.

## Как запустить проект

### 1. Клонировать репозиторий
```bash
git clone https://github.com/your-username/Flutter_Lab2.git
cd Flutter_Lab2/first_flutter_app
```
### 2. Получить зависимости
```bash
flutter pub get
```
### 3. Запустить в браузере Chrome
```bash
flutter run -d chrome
```
```
Приложение автоматически откроется в новой вкладке браузера. Для горячей перезагрузки используйте клавишу r в терминале, для полного перезапуска — R.
```
### Что изучено в работе
- Создание и структура Flutter-проекта (flutter create).

- Базовые виджеты: MaterialApp, Scaffold, Container, Center, Text.

- Стилизация текста через TextStyle и оформление фона через BoxDecoration.

- Линейный градиент (LinearGradient) с указанием направления (begin / end).

- Инструменты VS Code для Flutter: CodeLens, рефакторинг «Wrap with Widget», Flutter Inspector.

- Различие между Hot Reload (r) и Hot Restart (R).

- Инициализация Git-репозитория и работа с .gitignore для Flutter-проекта.
